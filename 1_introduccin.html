1. Introducción

Introducción

Hoy en día entre las bases de datos más utilizada, están el modelo de base de datos relacional. Este tipo de base de datos es una herramienta empleada por todo tipo de empresas, negocios u organización, independientemente de su tamaño o rubro al que se dedique. En este tema se ahondará un poco más sobre sus principales características y cómo funciona, ventajas y desventajas en las bases de datos relacionales, ya que en el primer tema de esta asignatura se definió el concepto y diferencia entre estas y las bases de datos no relacionales.

Objetivos

  1. Conocer los aspectos fundamentales de las bases de datos relacionales.
  2. Conocer la estructura de una base de datos relacional.
  3. Comprender el funcionamiento de las bases de datos relacionales.
